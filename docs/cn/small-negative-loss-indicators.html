小负损原因分析指标体系

小负损概述

小负损是指台区线损率持续处于轻微负值状态的现象，通常指连续60天以上线损率在-3%至0%之间。小负损虽然数值较小，但长期存在会导致电网管理与考核问题，需要找出并解决根本原因。

本指标体系通过设计一系列与小负损密切相关的诊断指标，为快速定位和解决小负损问题提供数据支持。

小负损可能原因及指标体系

基础判定指标

台区小负损持续天数 (TQ_SMALL_NEGATIVE_LOSS_DAYS)
台区线损率连续处于小负损范围（-3%至0%）的天数，超过60天即判定为长期小负损
COUNT(DISTINCT STATIS_DATE) WHERE -3% < LOSS_RATIO < 0%
某台区连续小负损天数统计示例
线损率曲线
小负损区域(-3%至0%)

原因一：CT倍率互感器异常

CT小负损异常指标(总表) (GM_CT_SMALL_NEGATIVE_ABNORMAL)
总表CT倍率异常导致小负损评分，评估CT倍率变更前后供用电比值变化
AVG(NEW_RATE/OLD_RATE) WHERE CT_PT_TYPE = 'CT' AND IsGM = 1 AND -3% < LOSS_RATIO < 0%
总表CT倍率变更前后供用电比值变化示例
变更前线损率(正值)
变更后线损率(小负值)
CT小负损异常指标(分表) (SM_CT_SMALL_NEGATIVE_ABNORMAL)
分表CT倍率异常导致小负损评分，评估CT倍率变更前后用电量变化
AVG(NEW_RATE/OLD_RATE) WHERE CT_PT_TYPE = 'CT' AND IsGM = 0 AND -3% < REGION_LOSS_RATIO < 0%
分表CT倍率变更前后用电量变化示例
变更前用电量趋势
变更后用电量趋势(轻微异常)

原因二：台区户表关系错误

户表关系小负损指标(总表) (GM_RELATION_SMALL_NEGATIVE_ERROR)
总表关联户表错误导致小负损程度，通过分析供用电相关性评估
CORR(CONSUMED, SUPPLIED) WHERE REGION_ID AND IsGM = 1 AND -3% < LOSS_RATIO < 0%
台区供用电相关性对比示例
供电量曲线
用电量曲线
小负损区域(用电量略大于供电量)
户表关系小负损指标(分表) (SM_RELATION_SMALL_NEGATIVE_ERROR)
分表关联台区错误导致小负损程度，通过分表用电量与台区供电量比值评估
SUM(Usage) / RELATED_REGION_SUPPLIED WHERE AreaId AND IsGM = 0 AND -3% < REGION_LOSS_RATIO < 0%
分表用电量与台区供电量比值示例
正常台区(比值<1.00)
小负损台区(比值1.00-1.03)
大负损台区(比值>1.03)

原因三：分布式能源反向计量异常

分布式能源小负损指标(总表) (GM_DER_SMALL_NEGATIVE_METERING)
总表分布式能源反向计量导致小负损评分，通过反向电量占比评估
SUM(Reversed_Energy) / SUPPLIED WHERE IsGM = 1 AND Has_Distributed_Generation = 1 AND -3% < LOSS_RATIO < 0%
分布式能源反向电量占比与线损率关系示例
正损台区(反向电量占比<1%)
小负损台区(反向电量占比1%-3%)
大负损台区(反向电量占比>3%)
分布式能源小负损指标(分表) (SM_DER_SMALL_NEGATIVE_METERING)
分表分布式能源反向计量导致小负损评分，通过单向计量设置评估
COUNT(Reversed_Energy > 0 AND Energy_Direction_Setting = 'UNIDIRECTIONAL') / COUNT(Has_Distributed_Generation = 1) WHERE IsGM = 0 AND -3% < REGION_LOSS_RATIO < 0%
分布式能源电表计量设置统计示例
大负损区域(线损率<-3%)
小负损区域(线损率-3%至0%)

原因四：时钟同步偏差

时钟同步小负损指标(总表) (GM_CLOCK_SMALL_NEGATIVE_DEVIATION)
总表时钟同步偏差导致小负损程度，通过时钟偏差平均值评估
AVG(ABS(Time_Diff)) WHERE IsGM = 1 AND -3% < LOSS_RATIO < 0%
总表时钟偏差与线损率关系示例
正常台区(偏差<5分钟)
小负损台区(偏差5-10分钟)
大负损台区(偏差>10分钟)
时钟同步小负损指标(分表) (SM_CLOCK_SMALL_NEGATIVE_DEVIATION)
分表时钟同步偏差导致小负损程度，通过中等偏差电表占比评估
COUNT(5 < ABS(Time_Diff) < 10) / COUNT(*) WHERE IsGM = 0 AND -3% < REGION_LOSS_RATIO < 0%
分表时钟同步偏差热力图示例
少量电表(0-10%)
中等电表(10-20%)
大量电表(20-50%)
绝大多数电表(>50%)
